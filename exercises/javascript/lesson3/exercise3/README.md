# JavaScript > Lesson3 > Exercise3

Ecrire une fonction qui donne la factorielle du nombre donné en paramètre.

_PS : une factorielle (aussi écrite n!) est la multiplication de tous les nombres entre 1 et n._

Exemples :

* 0! = 1
* 1! = 1
* 2! = 1 × 2 = 2
* 3! = 1 × 2 × 3 = 6
* 4! = 1 × 2 × 3 × 4 = 24
* 10! = 3628800
